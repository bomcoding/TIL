## Linux기초
* mkdir -p(--parents) : 필요하면 부모 디렉토리를 생성한다.
* sudo : (super user do)의 약자. 관리자 권한으로 명령어를 실행시킬때.
* nano 에디터 : 파일을 생성 및 수정 할 수 있는 에디터
* 패키지 매니저 : apt
* git : 버전관리 프로그램
